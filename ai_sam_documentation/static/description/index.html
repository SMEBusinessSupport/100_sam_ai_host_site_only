SAM AI Documentation & Insights

Knowledge Publishing System Built on Odoo eLearning

Transform your markdown documentation into structured eLearning courses automatically.

Key Features

Markdown to eLearning

Markdown files auto-convert to eLearning content on module upgrade.

Hierarchical Navigation

Folder structure becomes course sections with sidebar navigation.

Stable URLs

/sam_insights/ URLs remain stable even when content moves.

How It Works

Folder Structure eLearning Result
docs/00_course_folder/ eLearning Course (slide.channel)
docs/00_course_folder/section/ Section Header (is_category=True)
docs/00_course_folder/section/lesson.md Lesson Content (article slide)

Backend-Driven Documentation

Update markdown files, push to git, upgrade the module - your live documentation updates automatically. No database-driven content editing required.
